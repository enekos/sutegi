# Perf gate: accepted regressions.
#
# The aatxe perf gate compares a PR against master and fails on any regression
# past the noise gate. This file lists benches whose regression is *accepted* —
# either an intended cost we've decided to pay, or a bench whose inter-run
# variance the two-run comparison can't separate from signal. CI still reports
# their delta; it just doesn't fail on them. Everything not listed here still
# hard-fails, so the gate keeps its teeth on the stable micro-benches.
#
# Format: one bench name per line. `#` starts a comment. Keep a rationale.

# that irreducible cost; the rest is the feature. Accepted as the new baseline.
http_parse_request
